The Editor is no longer included by default, it is no longer Windows-only, you can
download the Editor for your platform from the releases page:

https://github.com/ZehMatt/7dtd-WalkerSim2/releases/tag/{version}

- For Windows download: WalkerSim2-Editor-Windows-x64-{version}.zip

- For Linux download: WalkerSim2-Editor-Linux-x64-{version}.zip

- For Mac download: WalkerSim2-Editor-Mac-arm64-{version}.zip

For older versions you can visit https://github.com/ZehMatt/7dtd-WalkerSim2/releases to
see all releases, but the latest release will always be available at the link above.

If you only want to change the amount of zombies you can edit the config file (WalkerSim.xml) directly,
adjust the number between <PopulationDensity> and </PopulationDensity>.
