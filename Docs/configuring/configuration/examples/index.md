# Examples

These are ready-to-use configuration presets that demonstrate different movement behaviors. Each example can be imported directly into the editor or copied into your `WalkerSim.xml` configuration file.

- [Classic Boids](classic-boids.md) - Traditional flocking behavior with cohesion, alignment, and separation
- [City Migrators](city-migrators.md) - Agents travel between cities along roads
- [Loose Swarm](loose-swarm.md) - Multiple groups drifting in different directions
- [Road Horde](road-horde.md) - Tight groups marching along roads
